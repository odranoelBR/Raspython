Básico

Delimitador de instrução ;

Delimitador de blocos { }

Para que seu código fique mais organizado e legível e aconselhado pular linha para cada instrução e apos os delimitadores de bloco. Utilize o ; para separar cada instrução. Visualize exemplos de como fazer.

Comando de movimento

CIMA - Movimenta o zumbi para cima.

BAIXO - Movimenta o zumbi para baixo.

DIREITA - Movimenta o zumbi para para direita.

ESQUERDA - Movimenta o zumbi para esquerda.

Exemplos : cima; baixo;

Estrutura condicional

Se (Condição booleana) { Bloco de execução};

Se (Condição booleana) { Bloco de execução}; Senao {Bloco de execução}

Se a Condição booleana for verdadeira então o Bloco de execução será executado.

Exemplo : Se (sensorfrente) { cima };

Condições booleanas

Verdadeiro

Falso

Sensorcima - Verifica se existe obstãculo em cima

Sensorbaixo - Verifica se existe obstãculo em baixo

Sensordireita - Verifica se existe obstãculo na direita

Senoresquerda - Verifica se existe obstãculo na esquerda

Negacao - Você pode negar qualquer booleano com ! antes da condiçõ como no exemplo.

Exemplo : Se (sensorfrente) { cima };

Exemplo : Se (!falso) { baixo };

Bloco de execução

Dentro dos blocos de execuçõ podem ser escritos varias instruções.

Não é necessãrio usar o separador ; para cada uma.

Exemplo:

Se (sensorcima) { cima baixo direita direita};

Enquanto(!falso) faca { se (sensorcima) { cima } };

Iterações

Repita NUMERO vezes { Bloco de execução };

Enquanto (Estrutura condicional) faca { Bloco de execução };

Exemplo : repita 5 vezes {cima};

Enquanto(!falso) faca { se (sensorcima) { cima } };
